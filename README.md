# Tic Tac Toe Game - Android Studio
This is a simple Tic Tac Toe game built on Android Studio using Java.

#How to Play
>Open the game on your Android device.
>You will see a 3x3 grid.
>The first player is X and the second player is O.
>The first player (X) places their symbol in an empty square by tapping on it.
>The second player (O) then places their symbol in an empty square.
>Players continue to take turns until one player gets three symbols in a row (horizontally, vertically, or diagonally), or until all squares are filled.
>If one player gets three symbols in a row, they win! If all squares are filled and no player has three in a row, the game is a tie.

#Features
>Two-player game
>Reset after a game ends (win or tie)

#Requirements
>Android device running Android 5.0 or newer
>Android Studio

#Installation
>Download the project as a ZIP file or clone the repository.
>Open the project in Android Studio.
>Build and run the project on your Android device.
